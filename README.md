# TClock

- An ASCII digital clock in the terminal
![](./assets/screenshot.png)

## Installation

1. Clone the repo
```sh
$ git clone git@github.com:NoSpawnn/TClock.git
$ cd TClock
```

2. Build and run
```sh
# Build then run
$ make
$ ./tclock

# Build and run
$ make run
```

3. Add it to your path if you want
```sh
$ cp ./tclock ~/.local/bin
```

## Usage

- Run the exectuable, quit with <kbd>Ctrl+C</kbd> or <kbd>q</kbd>
